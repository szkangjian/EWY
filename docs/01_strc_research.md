# STRC 深度研究：发行机制与价格稳定性分析

## 一、STRC 是什么

**全称**：Strategy Variable Rate Series A Perpetual Stretch Preferred Stock  
**发行方**：Strategy Inc（前 MicroStrategy）  
**交易所**：NASDAQ  
**面值（Par Value）**：**$100/股**  
**性质**：永续优先股（无到期日），可变利率，月度现金分红

> [!IMPORTANT]
> STRC 本质是一个**类债券的股权工具**：它表面上是优先股，但设计目标是模拟固定收益产品，为 Strategy 的比特币收购计划融资。

---

## 二、发行机制

### IPO（2025年7月）
- 发行 **2,800 万股**，定价 **$90/股**（折价发行）
- 募资约 **$24.7 亿**
- 2025年7月29日交割

### ATM（At-The-Market）持续增发
- 2025年7月31日启动，额度 **$42 亿**
- 根据市场价格和交易量灵活卖出新股
- 收入用于购买比特币和运营资金

---

## 三、核心机制：可变股息率如何让价格锚定 $100

STRC 的核心设计就是通过**动态调整股息率**来将价格拉回 $100 面值。

### 股息调整规则（基于 5 日 VWAP）

| 5日VWAP | 操作 | 逻辑 |
|---------|------|------|
| **< $95** | 股息率 **+50bp 或更多** | 大力刺激买入 |
| **$95 ~ $98.99** | 股息率 **+25bp 或更多** | 温和刺激需求 |
| **$99 ~ $100.99** | **不调整** | 价格处于目标区间 |
| **≥ $101** | 股息率 **-25bp 或更多** | 抑制过度溢价 |

### 股息率限制（SOFR 锚定）
- 下调幅度不得超过 **25bp + 当期 SOFR 变动**
- 股息率不得低于 **1个月 SOFR**（利率下限保护）
- 当前年化股息率：**11.50%**（2026年3月，已连续 7 次上调）

### 工作原理示意

```
价格跌到 $97 → 董事会加息 → 年化收益更高 → 吸引买入 → 价格回升 → 接近 $100
价格涨到 $102 → 董事会降息 → 收益率下降 → 卖出压力 → 价格回落 → 接近 $100
```

> [!TIP]
> 这个机制类似于央行的利率调控：通过调节"收益率"来控制"价格"，让 STRC 表现得像一个**货币市场基金**而非股票。

---

## 四、价格应该围绕 $100 波动吗？

**是的，$100 是设计目标价**，但实际表现：

| 时期 | 价格区间 | 原因 |
|------|---------|------|
| 2025年7月（IPO） | $90 → ~$95 | 折价发行，逐步爬升 |
| 2025年8月~11月 | $87 ~ $95 | 新产品，市场信心建立中 |
| 2025年底 | 短暂跌破 $97 | 比特币波动 + 市场不确定性 |
| 2026年1月~3月 | $99 ~ $100 | 多次加息后趋近目标 |

从你的数据看：**价格从 $95 逐步收敛到 $99.5，正在接近 $100 面值**——股息调节机制在起作用。

## 五、影响 STRC 价格偏离 $100 的因素

### 🔴 下行风险（价格 < $100）

| 因素 | 影响机制 |
|------|---------|
| **比特币大跌** | Strategy 资产负债表恶化 → 市场担忧分红能力 → 抛售 |
| **分红支付能力存疑** | Strategy 无实际运营收入，靠融资维持分红，若融资困难则风险暴露 |
| **市场利率上升** | 无风险利率上升 → STRC 相对收益率不够吸引 → 价格下跌 |
| **市场信心崩溃** | 若长期低于 $95 → 投资者视为信用风险而非稳定收益产品 |
| **ATM 稀释** | 持续增发新股 → 供给增加 → 压制价格 |

### 🟢 上行因素（价格 > $100）

| 因素 | 影响机制 |
|------|---------|
| **比特币牛市** | Strategy 资产大幅增值 → 分红确定性增强 → 买入推动溢价 |
| **市场利率下降** | 11.5% 收益率更具吸引力 → 需求增加 |
| **避险需求** | 股市大跌时资金涌入类固收产品 |

> [!WARNING]
> STRC 的分红**不由比特币直接担保**。Strategy 公司破产时，STRC 持有人在普通股东之前受偿，但在债权人之后。这不是无风险产品。

---

## 六、关键结论

1. **$100 是锚定目标**，STRC 的整个设计就是为了让价格围绕 $100 波动
2. **当前 $99.5 非常接近目标**，说明机制在有效运作
3. **短期偏离是正常的**（$95~$101 范围内），股息率会自动修正
4. **最大风险是比特币崩盘**导致 Strategy 无法维持分红，此时机制可能失效
5. **本质上是"用比特币波动率换取的高收益固收产品"**——适合做稳定收益策略，但不是零风险
