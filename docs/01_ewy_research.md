## 三、关键因子分析结果

### 3.1 Bid-Ask Spread（流动性）

| 指标 | 数值 |
|------|------|
| 分钟级中位波幅 | $0.02 |
| 中位波幅占价格比 | 0.026% |
| 零波幅分钟占比 | 11.6% |
| 日均成交量 | 511 万股 |
| 日均成交额 | ~$3.9 亿 |

> **结论**：EWY 流动性极好，spread 仅 $0.02，交易成本几乎可忽略。1000 股买卖一轮总成本约 $50（含 IB Pro 佣金）。

### 3.2 波动率特征

| 指标 | 数值 |
|------|------|
| 日内波动率中位数 | 1.46% |
| 年化波动率均值 (20日滚动) | 29.0% |
| 当前年化波动率 | **70.3%**（极高） |
| 最大单日跌幅 | **-11.31%** (2026-03-03) |
| 最大单日涨幅 | +10.31% |

> **结论**：EWY 波动率远高于 STRC（STRC 日波动 ~$0.50 即 0.5%，EWY 日波动 1.5~2%）。当前波动率处于历史极高水平（70%），暗示近期大幅波动。

### 3.3 极端下跌事件

| 统计 | 数值 |
|------|------|
| 日跌幅 > 2% 天数 | 55 天 (10.6%) |
| 日跌幅 > 3% 天数 | 26 天 |
| 日跌幅 > 5% 天数 | 8 天 |

**恢复模式**：
- 55 次大跌中，约 **70% 在 1~3 天内恢复**
- 但有 **8 次 30 天内未恢复**（多与系统性事件相关）
- 2025-04-08 跌 -5.04% 至 $48.78（全数据最低），次日即大幅反弹
- 2026-03-03 跌 -11.31%（$146 → $130），至今未恢复

### 3.4 除息日行为（Dividend Capture 不可行）

| 除息日 | 股息 | 实际跳空 | Cushion |
|--------|------|---------|---------|
| 2024-12-17 | $1.298 | -$0.20 | **+$1.098** |
| 2025-12-16 | $2.037 | -$2.71 | **-$0.673** |

5 年平均 Cushion: **-$0.134**（跳空大于股息）

> **结论**：EWY 不具备 STRC 那样的 Dividend Capture 条件。没有面值锚定机制，除息日跳空完全受市场情绪驱动，无可预测的"缓冲垫"。

### 3.5 相关性分析

| 对比标的 | 相关系数 | Beta | 含义 |
|---------|---------|------|------|
| **EEM** (新兴市场) | **0.807** | 1.353 | 高度正相关，EWY 是 EM 的放大器 |
| **SMH** (半导体) | 0.612 | 0.506 | 中等正相关，三星/SK 海力士驱动 |
| **SOXX** (半导体) | 0.627 | 0.503 | 中等正相关 |
| **SPY** (美国大盘) | 0.565 | 1.044 | 中等正相关，Beta ≈ 1 |
| **USD/KRW** (汇率) | 0.154 | 0.471 | 弱正相关 |

> **结论**：EWY 的主要驱动因子是**新兴市场整体情绪** (EEM 相关 0.81) 和**半导体周期** (SMH 相关 0.61)。韩元汇率的直接相关性较弱（0.15），可能因为时区错位（韩国市场先于美国）。

### 3.6 收益率分布

| 指标 | 数值 |
|------|------|
| 日均收益率 | 0.159% (年化 40.0%) |
| 日收益率标准差 | 1.90% (年化 30.2%) |
| 偏度 | -0.113（略左偏） |
| 峰度 | 3.666（厚尾） |
| 正收益天数比例 | 53.9% |
| Sharpe Ratio (Rf=4.5%) | **1.18** |

### 3.7 成交量模式

成交量在美东时间 **下午 3:00-4:00（20:00 UTC）** 最集中，这是收盘前的高峰。开盘前 30 分钟仅占 0.3% 的成交量。除息日成交量无明显异常（与 STRC 完全不同）。

## 五、初步策略方向建议

基于以上因子分析，EWY 适合的策略方向（待进一步验证）：

1. **趋势跟踪 / 动量策略** — EWY 有明显的趋势性（2025 年从 $48 涨到 $152），动量因子可能有效
2. **均值回归（短期）** — 大跌后 1~3 天内恢复的概率较高（~70%），可研究"跌 >3% 买入，反弹 2% 卖出"
3. **半导体周期轮动** — 利用 EWY 与 SMH 的相关性，在半导体上行周期增配 EWY
4. **波动率交易** — 当前波动率 70% 处于极端水平，可能存在均值回归机会

### 不建议的策略
- ❌ Dividend Capture — 数据已证明不可行
- ❌ 窄幅区间交易 — 波动率太高，无法设定稳定的买卖区间
