## 1. 产品概要：STRC 是什么

| 项目 | 内容 |
| ---- | ---- |
| 全称 | Strategy Variable Rate Series A Perpetual Stretch Preferred Stock |
| 发行方 | Strategy Inc（前 MicroStrategy） |
| 面值 | $100 |
| 股息 | 月度可变，当前年化 11.50%（~$0.96/月/股） |
| 税务分类 | **100% Return of Capital (ROC)** — 当期不交税，降低 cost basis |
| 价格锚定 | 股息率动态调节 + ATM 增发 → 围绕 $100 波动 |
| 价格上限 | ~$100.3（ATM 增发硬顶） |
| 底层资产 | Strategy Inc. 的资产负债表，核心是其持有的 BTC |

STRC 本质上是一只**挂钩比特币的高息优先股**。发行方 Strategy 通过不断购入 BTC 累积资产，再以优先股的形式向投资者派发浮动高息。其价格被 $100 面值锚定，波动极小（日常在 $99~$101 之间），但它的股息率（年化 ~11.5%）远高于市场上任何固收产品。

## 2. 策略架构：三层收益引擎

```text
┌─────────────────────────────────────────┐
│     第一层：底仓（长期持有，永不卖出）     │
│  → 年化 11.5% ROC 股息                   │
│  → 不卖不交税，无限期税务递延              │
├─────────────────────────────────────────┤
│    第二层：Dividend Capture（每月 1 次）   │
│  → 除息前一天买入（动用闲置 Margin）       │
│  → 除息日开盘卖出                         │
│  → 捕获 ~$0.60~$0.80/股 净利              │
├─────────────────────────────────────────┤
│     第三层：极端事件加仓（BTC 闪崩时）     │
│  → STRC < $95 时用预留现金加仓             │
│  → 恢复 $99+ 后卖出或并入底仓              │
└─────────────────────────────────────────┘
```

## 3. 底仓规则：长期持有与建仓时机

### 3.1 建仓

- 目标价位：**$98.5~$100**
- 分批建仓：分 3~5 天买入，避免单日大额影响
- **建仓绝佳时机 (U型谷底)**：**除息日后的第 2 到第 3 天 (T+2 / T+3)**。此时刚刚走完由隔夜套利客引发的"除息抛压踩踏"，价格往往处于当月最低谷（除息后遗症谷底）。绝不要在宣告日(T0)或者除息日当天(T+0)早盘去接飞刀。

### 3.2 持有

- **永不卖出** — ROC 股息只要不卖就不产生税务事件
- 每月收到 ~$0.96/股现金
- Cost basis 每月自动降低 $0.96

### 3.3 Cost Basis 追踪

以 $100 买入 1000 股为例：

| 月份 | 累计 ROC | Cost Basis | 注意事项 |
| ---- | -------- | ---------- | -------- |
| 第 1 月 | $0.96 | $99.04 | |
| 第 6 月 | $5.76 | $94.24 | |
| 第 12 月 | $11.52 | $88.48 | |
| 第 24 月 | $23.04 | $76.96 | |
| basis → $0 | ~第 104 月 | $0 | 此后 ROC 变为 capital gain |

> [!NOTE]
> Basis 归零约需 **8.7 年**（按当前利率）。Strategy 预计 ROC 处理至少维持 **10 年**。

## 4. Dividend Capture 规则：每月套利操作

### 4.1 操作日历（每月执行）

| 步骤 | 时间 | 操作 |
| ---- | ---- | ---- |
| **监控** | 月初/月末 | 等待 GitHub Actions 哨兵推送的自动报警邮件 |
| **预检** | 除息日前 2 天 | 检查 BTC 是否稳定、STRC 价格是否正常 |
| **买入** | 除息日前 1 天（3:50 PM） | 买入目标数额（视闲置融资而定，不局限于底仓大小） |
| **卖出** | 除息日当天（9:31~9:35 AM） | 开盘后立即卖出 |
| **记录** | 卖出后 | 记录买卖价、股息、净利 |

### 4.2 下单细节（手动接管时）

- **买入**：前日 3:50 PM，用 **Limit Order @ 市价 +$0.02**，确保成交
- **卖出**：除息日 9:31 AM，用 **Market Order** 或 **Limit @ 市价 -$0.02**
- 确保账户设置为 **LIFO 或 Specific Identification** 卖出方式（绝对不要用 FIFO，避免洗掉底仓）

### 4.3 历史回测（6 次全部盈利）

| 除息日 | 买入 | 卖出 | 股息 | 净利/股 |
| ------ | ---- | ---- | ---- | ------- |
| 2025-10-15 | 98.92 | 98.80 | $0.854 | **+$0.71** |
| 2025-11-14 | 99.18 | 98.80 | $0.875 | **+$0.48** |
| 2025-12-15 | 99.02 | 98.74 | $0.896 | **+$0.60** |
| 2026-01-15 | 100.30 | 100.20 | $0.917 | **+$0.80** |
| 2026-02-13 | 100.01 | 100.01 | $0.938 | **+$0.92** |
| 2026-03-13 | 100.01 | 99.82 | $0.958 | **+$0.75** |

**平均净利：$0.71/股，胜率 6/6 = 100%**

### 4.4 为什么存在这种套利空间？（市场微观结构）

你可能会问：**既然每次都能净赚 ~$0.70，为什么教科书说"除息日股价应精确下跌股息的金额"，而 STRC 开盘却没有暴跌？**

答案是：**专业机构确实在做，而且你看到的价格现象，正是他们集体行动的结果！**

1. **为什么除权跌幅远小于股息？（"缓冲垫 Cushion"效应）**
   理论上，除权日开盘确实应该直接扣掉 $0.96（假设前收盘 $100，开盘理应 $99.04）。但我们能在开盘以 $99.50+ 卖出的"缓冲垫 (Cushion)"，完全来自这股庞大的**逆向流动性**：
   - **高净值避税买盘**：故意避开 T-1 拿息（全额交所得税），专挑 T+0 买"打折股"，把分红转化为未来的长期资本利得（只需交 20%）。
   - **无脑 DRIP 机器**：红利自动再投资程序会在 T+0/T+1 无视基本面市价抢筹。
   - **Yield-on-Cost 囤币党**：为了追求更高的派息股数，逢低疯狂买入。

2. **既然机构在做，散户为什么还能吃肉？（流动性摩擦限制）**
   大机构资金（几千万级）的体量成为了高频 Capture 的诅咒。他们在 T-1 尾盘强买会把价格拉出溢价，第二天砸出恐慌盘又会把自己砸出深坑。而散户拿着千股进出，对市场深度完全是"隐形"的，我们是在巨鳄互殴的缝隙里"寄生套利"。

## 5. 风控铁律：基于派息变化的执行纪律 (The Payout Rule)

你只需通过手机查收每月初 GitHub Actions 云端哨兵推送的自动报警信。在收到报警的瞬间，其中的**派息金额相较于上个月的变化**，直接决定了本月套利的生死：

1. **🟢 加息月 (Hike, e.g. $0.94 → $0.96)**：**满仓出击**。市场预期高涨，T+0 买盘极强，你的跳空缓冲垫（Cushion）极厚，完美套利。
2. **🟡 平息月 (Unchanged)**：**正常出击（削减利润预期）**。市场缺少额外的超额预期，虽然仍然有 DRIP 等机器托底，但你的 Cushion 会变薄，净利润空间会被压缩。
3. **🔴 降息月 (Cut, e.g. $0.96 → $0.80)**：**【系统熔断】无条件放弃本月套利！千万不可动用杠杆！**
   - 降息会导致底层价值重估，T+0 的托盘大军瞬间叛逃甚至同向砸盘。
   - 真实的向下跳空缺口势必 **远大于** 你拿到的微薄股息。不仅赚不到差价，还会承受 Margin 利息+本金暴跌的重磅回撤。

## 6. 极端事件加仓规则

### 6.1 触发条件

- STRC 日内跌破 **$95**

### 6.2 历史极端事件

| 事件 | 最低点 | 恢复时间 |
| ---- | ------ | -------- |
| 2025-11-17 BTC 暴跌 | $92.47 | 12 天 |
| 2025-11-20 继续下探 | $90.68 | 3 天 |
| 2026-02-05 BTC 闪崩 | $93.00 | **1 天** |

### 6.3 执行规则

- 用预留的 **20~30% 现金** 分批加仓
- $95 买入 1/3，$93 买入 1/3，$91 买入 1/3
- 恢复到 **$99+** 后卖出（赚差价）或并入底仓（增加固收）
- **绝不用杠杆**

## 7. 风险控制与硬性纪律

### 7.1 风险矩阵

| 风险 | 概率 | 影响 | 应对 |
| ---- | ---- | ---- | ---- |
| BTC 短期闪崩 | 中 | 账面亏损 $3~$7/股 | 不止损，用预留现金加仓 |
| BTC 长期熊市 | 低 | 股息支付能力存疑 | 监控 Strategy 现金储备 |
| Strategy 降息 | 中 | 价格短期下跌 | 降息是价格到位的信号 |
| ATM 大量增发 | 中 | 供给压力 | 价格接近 $100 时增发反而是稳定信号 |
| ROC 政策变更 | 极低 | 股息变为 taxable | 至少 10 年内不变 |
| Capture 当晚闪崩 | 低 | 加仓部分亏损 | 单次最大亏损 ~$3/股 |

### 7.2 硬性纪律

1. **底仓永不卖出**（除非 Strategy 基本面恶化）
2. **底仓部分永不使用杠杆**（只在每月的 Dividend Capture 短期动用 Margin）
3. **保留 20~30% 的现金/国债缓冲池** 应对极端事件和做大盘定投
4. **Dividend Capture 份额【必须】在次日开盘 9:35 前全部清仓，哪怕亏损也必须割肉弃车！**
   - **核心逻辑**：用来做 Capture 的钱是加杠杆借来的大资金。你的目标是"赚利差"，而不是"炒币"。如果除息日开盘因大盘不佳导致浮亏，**绝对不能抱有"等回本再卖"的心态**。多持有一天，破产风险指数级上升。
5. 若除息前一天 BTC 正在暴跌（>5%），**无条件跳过本月 Capture**

### 7.3 监控指标

- Strategy 月度股息率公告
- Strategy 季度 8-K 报告（BTC 持仓量、现金储备）
- BTC 价格（日线级别）
- STRC 5 日 VWAP（是否在 $99~$101 区间）

STRC 的税务结构（100% ROC）赋予了它独特的税务优势。详细分析请参阅专题文档：

- 美国居民 → [06_strc_tax_wash_sale.md](06_strc_tax_wash_sale.md)（Wash Sale 斩链操作）

**速览对比**：

| 维度 | 美国居民 | NRA (中国居民) |
| ---- | -------- | -------------- |
| ROC 股息 | 免当期税，降 cost basis | **零预扣，100% 落袋** |
| Capture 差价 | 短期 CG（最高 37%） | **零美国税** |
| Wash Sale | 适用（30 天限制） | **不适用** |
| 年度操作 | 12 月斩链 + 1099 报税 | 保持 W-8BEN 有效即可 |

## 9. 自动化工具链

### 9.1 全自动云端哨兵 (GitHub Actions Sentinel)

你无需手动运行任何脚本。该系统已部署在 GitHub 服务器上，每天自动查询 Polygon API 获取最新的 STRC 分红数据。

- **触发通知**：一旦发现新的派息公告，你的手机 GitHub App / 绑定邮箱会立即收到 `🚨 STRC 股息公告触发` 的强提醒推送
- **行动向导**：推送正文会自动比对"增/减息数值"，并直接告知你是应该【满仓】、【正常】还是【熔断】

### 9.2 半自动买卖脚本 (`strc_ibkr_capture.py`)

通过代码连接 IB API，确保极速、带滑点保护的挂单执行。

- **为什么连 IB Gateway 而不是 TWS？**
  TWS 是笨重的交易界面（占用高，弹窗多，易卡死）。IB Gateway 是专为 API 和脚本设计的极简网关，长期运行更加稳定。
