# EWY 交易策略研究：修正口径后的结论

> **版本**: v2.0 | **最后更新**: 2026-04-01
>
> 本文基于仓库内分钟数据，但所有回测结果均统一采用 `US/Eastern` 时区，并只使用常规交易时段 `09:30-16:00 ET`。

---

## 一、这次修正了什么

旧版研究把一部分 `Polygon` 历史数据当成“无时区本地时间”直接按自然日聚合，而 `Yahoo` 更新数据又按美东时间写入。结果是：

- 部分盘后数据被归到了第二个交易日
- 日线 `Open/High/Low/Close` 被扩展时段污染
- `IBS` 和“相对前日收盘跌幅”这两个核心信号都被放大

修正后，仓库里的研究口径变为：

| 项目 | 数值 |
|------|------|
| 原始分钟数据 | `219,296` 行 |
| 统一到美东后常规时段分钟数据 | `191,397` 行 |
| 交易日数量 | `500` |
| 覆盖区间 | `2024-03-25` 到 `2026-03-23` |

这不是“小修小补”，而是会直接改变策略结论的底层修正。

## 二、策略一：IBS 均值回归

### 2.1 规则

- 买入：当日 `IBS < 0.2` 且 `Close > MA200`，收盘买入
- 卖出：当日 `IBS > 0.8`，收盘卖出
- 安全阀：最大持有 `10` 个交易日

`IBS = (Close - Low) / (High - Low)`

### 2.2 回测结果

| 指标 | 数值 |
|------|------|
| 交易笔数 | `28` |
| 胜率 | `71%` |
| 总收益 | `+33.3%` |
| 均收益/笔 | `+1.19%` |
| 最大单笔亏损 | `-5.18%` |
| 平均持有 | `3.25` 天 |
| 累计持有天数 | `91` 天 |
| 退出方式 | `28/28` 全部由 `IBS > 0.8` 正常退出 |

### 2.3 如何理解这个结果

- **IBS 仍然有效**。修正后它没有失效，说明 EWY 的短线均值回归不是完全由时区错误制造出来的。
- **但旧版文档明显高估了强度**。旧口径写的是 `83%` 胜率、`+62.0%` 总收益；修正后更接近“有 alpha，但不神奇”。
- **IBS 的优势仍然在信息密度**。它比单纯跌幅更能区分“跌很多但收在高位”和“跌很多且收在低位”。

### 2.4 结论

IBS 现在仍然是 EWY 上最值得保留的主策略，但应把它理解成：

- 一个建立在强势 regime 上的短线反弹因子
- 一个会经历回撤、并非 80%+ 胜率圣杯
- 一个需要持续接受 MA200 和样本外验证约束的信号

## 三、策略二：盘中跌幅触发反弹

### 3.1 参数扫描设置

- 入场跌幅：`-3%`、`-3.5%`、`-4%`、`-4.5%`、`-5%`
- 反弹目标：`+1%`、`+1.5%`、`+2%`、`+2.5%`、`+3%`
- 最大持有：`1`、`2`、`3`、`5` 天
- 止损：无、`-2%`、`-3%`、`-5%`

共扫描 `400` 种组合。

### 3.3 旧版主推参数现在怎么样

旧版最常引用的规则是 `-3% / +2.5% / 3天 / 无止损`。修正后结果变为：

| 指标 | 数值 |
|------|------|
| 交易笔数 | `30` |
| 胜率 | `63%` |
| 总收益 | `+1.0%` |
| 均收益/笔 | `+0.03%` |
| 最大亏损 | `-8.34%` |

这说明旧版文档里把它写成“成熟主策略”是不成立的。

### 3.4 持有天数与止损

固定 `入场 -4% / 反弹 +2% / 无止损`：

| 持有天数 | 笔数 | 胜率 | 总收益 | 均收益 | 最大亏损 |
|---------|------|------|--------|--------|----------|
| 1天 | 16 | 62% | -8.9% | -0.56% | -6.53% |
| 2天 | 16 | 44% | -1.5% | -0.09% | -5.03% |
| 3天 | 15 | 60% | -1.9% | -0.13% | -11.23% |
| 5天 | 15 | 67% | +9.4% | +0.63% | -5.79% |

固定 `入场 -4% / 反弹 +2% / 持有 3 天`：

| 止损 | 笔数 | 胜率 | 总收益 | 均收益 | 最大亏损 |
|------|------|------|--------|--------|----------|
| 无 | 15 | 60% | -1.9% | -0.13% | -11.23% |
| -2% | 16 | 38% | -6.5% | -0.41% | -2.00% |
| -3% | 16 | 44% | -7.7% | -0.48% | -3.00% |
| -5% | 16 | 56% | -0.7% | -0.04% | -5.00% |

### 3.5 结论

- 盘中跌幅反弹**没有完全失效**，但强度远弱于旧版文档。
- 如果继续研究，它更像**观察型辅助策略**，不是仓库的第一主策略。
- 当前样本内相对更好的版本是 `-3.5% / +2.0% / 5天`，但这仍然是样本内最优，不应被描述为“稳定最优”。

## 四、过滤器测试

以下结果基于旧版常用参数 `-3% / +2.5% / 3天`，目的是看过滤器能否救回这条策略。

| 过滤器 | 笔数 | 胜率 | 总收益 | 均收益 | 最大亏损 |
|--------|------|------|--------|--------|----------|
| 基准（无过滤） | 30 | 63% | +1.0% | +0.03% | -8.34% |
| +MA200 | 15 | 87% | +17.2% | +1.15% | -8.34% |
| +IBS<0.2 | 6 | 83% | +1.1% | +0.18% | -4.95% |
| +IBS<0.3 | 9 | 56% | -10.3% | -1.15% | -6.29% |
| +RSI-2<10+MA200 | 2 | 100% | +2.7% | +1.34% | +0.18% |
| +BB下轨 | 2 | 0% | -6.5% | -3.26% | -4.95% |
| +MA200+IBS<0.3 | 3 | 100% | +5.2% | +1.73% | +0.18% |

### 4.1 过滤器结论

- **MA200 仍然有价值**。它显著改善了跌幅策略的质量，说明 EWY 的反弹信号对 regime 非常敏感。
- **IBS 作为跌幅策略的二次过滤并不稳定**。样本一少就容易被偶然性支配。
- **RSI / BB 目前没有证明自己适合 EWY**。要么样本太少，要么结果直接变差。

## 五、哪些旧结论需要收回

### 5.1 需要收回

- “跌幅策略 `73%` 胜率、`+34.7%` 总收益，可以作为成熟主策略”
- “无论什么事件类型都一样，短期都会反弹”
- “连续触发自动形成网格效应，是策略优势”
- “IBS 在所有维度都显著碾压跌幅策略，而且强度非常高”

### 5.2 仍然成立，但要更克制

- EWY 在强势半导体/AI regime 中，确实存在可交易的短线均值回归现象
- IBS 比单纯跌幅更有信息量
- MA200 更适合作为风控开关
- 止损在这类短线反弹策略里未必提升收益

## 六、当前推荐框架

| 角色 | 当前定位 |
|------|----------|
| IBS | 主策略 |
| 盘中跌幅反弹 | 观察型辅助策略 |
| MA200 | 必留的风险开关 |
| 事件归因 | 事后解释有价值，但不应替代规则 |

更准确的总结是：

> 修正时区和常规交易时段后，EWY 的短线均值回归效应仍然存在，但旧版文档明显高估了策略强度。IBS 可继续作为主策略研究与执行，盘中跌幅反弹只适合作为辅助策略，并且必须承认样本内优化和 regime 依赖的风险。
