## 二、策略一：IBS 均值回归

### 2.1 规则

- 买入：当日 `IBS < 0.45` 且 `Close > MA200`，收盘买入（v2 参数）
- 卖出：当日 `IBS > 0.90`，收盘卖出
- 安全阀：最大持有 `10` 个交易日（约 2 周）

`IBS = (Close - Low) / (High - Low)`

### 2.2 参数演化

#### v1：0.20/0.80（来自 0.1 步长扫描）

| 指标 | 数值 |
|------|------|
| 交易笔数 | 30 |
| 胜率 | 70% |
| 总收益 | +37.2% |
| 均收益/笔 | +1.24% |

#### v2：0.45/0.90（来自 0.05 步长精扫 + walk-forward 验证）

| 指标 | 数值 |
|------|------|
| 交易笔数 | **42** |
| 胜率 | **81%** |
| 总收益 | **+93.6%** |
| 均收益/笔 | **+2.23%** |
| 最大单笔亏损 | -4.0% |

#### 0.05 步长热图（max_hold=10, MA200, 总收益%/笔数）

| buy / sell | 0.70 | 0.75 | 0.80 | 0.85 | 0.90 |
| ---------- | ---- | ---- | ---- | ---- | ---- |
| 0.20 | +50.2(33) | +48.1(31) | **+37.2(30)** | +40.6(27) | +48.5(25) |
| 0.30 | +49.5(37) | +47.4(35) | +42.7(34) | +46.1(31) | +56.2(29) |
| 0.35 | +53.9(42) | +51.7(40) | +58.0(39) | +65.4(36) | +75.1(33) |
| 0.40 | +61.8(47) | +59.7(45) | +65.9(44) | +73.3(41) | +82.3(37) |
| **0.45** | +70.2(53) | +68.1(51) | +77.0(50) | +84.5(47) | **+93.6(42)** |

整个 `buy ≥ 0.30, sell ≥ 0.85` 区域单调向右上递增 — 不是孤峰。

### 2.3 Walk-forward 验证

把 322 个可交易日（去掉 200 天 MA 预热）切两半，分别跑同一网格：

| 时段 | 区间 | 行情 | v1 (0.20/0.80) | v2 (0.45/0.90) |
| ---- | ---- | ---- | -------------- | -------------- |
| H1 | 2025-01-10 → 2025-09-02 | +34% | 12笔, 67%, +6.6% | **12笔, 83%, +14.7%** |
| H2 | 2025-09-03 → 2026-04-23 | +107% | 18笔, 72%, +30.6% | **30笔, 80%, +79.0%** |

**反向验证**：H1 最优 = H2 最优 = `0.45/0.90`。这是 walk-forward 教科书式的"全周期一致性"信号。

详见 [`ewy_walkforward.py`](../../EWZ/ewy_walkforward.py)。

### 2.4 结论

IBS 是 EWY 上最有效的主策略。v1 → v2 升级的关键认知：
- **0.1 步长 grid 漏掉了真实最优**：[0.10, 0.15, 0.20, 0.25, 0.30] 直接跳过 0.45。
- **更宽入场不是过拟合**：在温和上行（H1，年化~50%）和强势上行（H2，年化~200%）两段都是 top 1。
- **MA200 总闸足够保护**：v2 入场更宽不意味更危险——所有交易仍在趋势之上。
- **唯一 caveat**：522 天数据全部在 EWY 牛市期，没真实熊市样本。如果韩国进入持久下跌，应回归到 v1 或暂停策略。

## 三、策略二：盘中跌幅触发反弹

### 3.1 参数扫描设置

- 入场跌幅：`-3%`、`-3.5%`、`-4%`、`-4.5%`、`-5%`
- 反弹目标：`+1%`、`+1.5%`、`+2%`、`+2.5%`、`+3%`
- 最大持有：`1`、`2`、`3`、`5` 个交易日
- 止损：无、`-2%`、`-3%`、`-5%`

共扫描 `400` 种组合。

### 3.3 宽松参数对比

参考组合 `-3% / +2.5% / 3个交易日 / 无止损` 表现较弱：

| 指标 | 数值 |
|------|------|
| 交易笔数 | `33` |
| 胜率 | `64%` |
| 总收益 | `-3.2%` |
| 均收益/笔 | `-0.10%` |
| 最大亏损 | `-8.34%` |

入场阈值 `-3%` 过浅，噪音交易太多，总收益已经为负。

### 3.4 持有天数与止损

固定 `入场 -4% / 反弹 +2% / 无止损`：

| 持有天数 | 笔数 | 胜率 | 总收益 | 均收益 | 最大亏损 |
|---------|------|------|--------|--------|----------|
| 1个交易日 | 20 | 65% | -6.9% | -0.34% | -6.53% |
| 2个交易日 | 20 | 50% | +3.2% | +0.16% | -5.03% |
| 3个交易日 | 18 | 61% | -3.0% | -0.17% | -11.23% |
| 5个交易日 | 18 | 72% | +15.4% | +0.86% | -5.79% |

固定 `入场 -4% / 反弹 +2% / 持有 3 个交易日`：

| 止损 | 笔数 | 胜率 | 总收益 | 均收益 | 最大亏损 |
|------|------|------|--------|--------|----------|
| 无 | 18 | 61% | -3.0% | -0.17% | -11.23% |
| -2% | 20 | 40% | -6.5% | -0.33% | -2.00% |
| -3% | 19 | 42% | -11.7% | -0.61% | -3.00% |
| -5% | 19 | 58% | -1.7% | -0.09% | -5.00% |

### 3.5 结论

- 盘中跌幅反弹有正收益，但强度远弱于 IBS，适合做**辅助策略**。
- 当前样本内最优参数是 `-4.5% / +2.5% / 5个交易日`，最新样本为 `12` 笔交易，实盘仍应对样本外衰减保持充分预期。

## 四、过滤器测试

以下结果基于宽松参数 `-3% / +2.5% / 3个交易日`，测试不同过滤器对跌幅策略的改善效果。

| 过滤器 | 笔数 | 胜率 | 总收益 | 均收益 | 最大亏损 |
|--------|------|------|--------|--------|----------|
| 基准（无过滤） | 33 | 64% | -3.2% | -0.10% | -8.34% |
| +MA200 | 18 | 83% | +13.0% | +0.72% | -8.34% |
| +IBS<0.2 | 6 | 83% | +1.1% | +0.18% | -4.95% |
| +IBS<0.3 | 10 | 50% | -16.4% | -1.64% | -6.29% |
| +RSI-2<10+MA200 | 2 | 100% | +2.7% | +1.34% | +0.18% |
| +BB下轨 | 2 | 0% | -6.5% | -3.26% | -4.95% |
| +MA200+IBS<0.3 | 4 | 75% | -0.9% | -0.23% | -6.09% |

### 4.1 过滤器结论

- **MA200 效果最显著**。加上 MA200 后，宽松跌幅策略从亏损转为正收益，说明 EWY 的反弹信号对趋势环境非常敏感。
- **IBS 作为跌幅策略的二次过滤不稳定**。样本一少就容易被偶然性支配。
- **RSI / BB 不适合 EWY**。要么样本太少，要么结果直接变差。

---

## 五、需要注意的局限

- 跌幅策略作为辅助有正收益，但不足以独立支撑
- 反弹效应依赖半导体/AI 强势周期，不能假设”什么原因跌都会弹”
- 连续触发时不应自动网格加仓，单笔持仓更安全
- 止损在这类短线反弹策略里未必提升收益

## 六、推荐框架

| 角色 | 定位 |
|------|------|
| IBS | 主策略 |
| 盘中跌幅反弹 | 观察型辅助策略 |
| MA200 | 必留的风险开关 |
| 事件归因 | 事后解释有价值，但不应替代规则 |

总结：

> EWY 在半导体/AI 强势周期中存在可交易的短线均值回归效应。IBS 是主策略，盘中跌幅反弹是辅助策略，两者都依赖趋势环境（MA200），且需承认样本内优化风险。

### 最新观察

- `2026-04-02` 当天，盘中跌幅策略触发并在日内达到 `+2.5%` 目标。
