## 一、ATM 天花板效应 — $100 是硬顶

稳定期数据证实了你的判断：

| 指标 | 数据 |
|------|------|
| 分钟价 ≥ $100 的时间 | 19.6% |
| 分钟价 ≥ **$100.5** 的时间 | **0.01%**（仅 3 个分钟） |
| 收盘价 ≥ $100.5 | **0 天** |
| 日内突破 $100 后的平均超越幅度 | **+$0.21** |

> [!IMPORTANT]
> **ATM 增发机制 = 卖方无限弹药**。价格一碰 $100，Strategy 就可以按市价卖新股，无限供给直接压制价格。这意味着 **$100~$100.3 就是实际天花板**，$101 高抛不现实。

日内触及 $100 的 40 天中，**超过 $100.5 的仅有 3 天**（11月初市场效率不足时）。到 2026 年后，价格几乎精准卡在 $100.0~$100.1。

## 三、除息日 — 每月 $0.96 的「地震」

年化 11.5% ÷ 12 = **每月 $0.96/股** 除息下跌

这意味着：
- 除息日前一天 $99.5 → 除息日开盘理论上变 **$98.54**
- 看起来像是「跌破支撑」，实际是正常除息
- **如果策略不避开除息日，会产生虚假买入信号**

稳定期中日内跌幅 >$0.50 的天数有 **13 天**，其中多数与除息相关。

## 五、还需要考虑的关键因素

### 1. 交易成本
- Bid-ask spread 约 **$0.02/股**
- $0.50 利润目标下，spread 吃掉 **4%**
- 免佣平台（Robinhood/Public）很重要

### 2. 除息日策略
- 必须跟踪每月除息日（record date / ex-date）
- 除息日前如果持仓，拿到 $0.96 股息但价格跌 $0.96 → 零和
- **建议**：除息日前不做新买入，等除息后的价格回稳

### 3. 股息率变更公告
- Strategy 每月初宣布新利率
- **加息 = 价格在低位的信号**（前瞻性买入指标）
- **降息 = 价格过高的信号**

### 4. 比特币闪崩风险
- 2025-11-17 单日跌 $3.74（$98.99 → $95.25）
- 2026-02-05 单日跌 $2.91
- 这些极端事件可以击穿任何止损

### 5. 资金效率
- $99.5 买 / $100 卖 = 每 $9,950 本金赚 $50 = **0.5% 收益/9天**
- 年化不错但**绝对金额小**，需要大仓位才有意义
