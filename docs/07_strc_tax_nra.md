# STRC 跨境税务指南：非美国税务居民 (NRA) 专题

> 本文档专门面向**持有中国护照、无美国绿卡**，通过 Interactive Brokers 等境外券商执行 STRC 策略的投资者。
> 美国税务居民请参阅 [06_strc_tax_wash_sale.md](06_strc_tax_wash_sale.md)。

---

## 一、NRA 身份的核心税务优势

对于非美国税务居民 (Non-Resident Alien, NRA)，STRC 策略在税务结构上拥有**碾压级的天然优势**：

| 收入类型 | 美国居民税务处理 | NRA 税务处理 |
| -------- | ---------------- | ------------ |
| ROC 股息（底仓收息） | 当期免税，但降低 cost basis | **零预扣，零申报** |
| Capture 差价利润 | 短期资本利得（最高 37%） | **零美国税** |
| Wash Sale Rule | 适用（30 天限制） | **不适用** |
| 年度报税义务 | 必须申报 (1040) | 仅需保持 W-8BEN 有效 |

---

## 二、逐项拆解

### 2.1 ROC 股息 → NRA 零预扣

STRC 的 distribution 被归类为 **Return of Capital (ROC)**，即"返回你自己的本金"，而**不是**美国税法下的 "dividend income"。

对 NRA 而言：
- **ROC 不属于美国来源收入 (US-source income)**
- 理论上 **不需要被预扣任何美国税款** — 0%
- 这比美国税务居民更有利（美国居民虽然也不交当期税，但 ROC 降低 cost basis，未来卖出时要交资本利得税）

> [!WARNING]
> **实操风险**：IB 的预扣系统有时可能会"先扣后退"——在年初发放时按 10% 先预扣（因为 IB 可能要等到年末的 1099 表才能确认 ROC 分类），然后年末需要通过 IB 税务调整退回。你需要在 IB 的 Tax Documents 里仔细核实每次分配的分类。

### 2.2 资本利得 → NRA 完全免税

非美居民买卖美股产生的**资本利得 (Capital Gains)**：
- **不需要交任何美国联邦税**（只要不构成 "Effectively Connected Income"，即你人不在美国开公司做这个事）
- Dividend Capture 中 T-1 买入 → T+0 卖出的差价利润，在美国端 = **0 税**

### 2.3 Wash Sale Rule → NRA 不适用！

美国居民必须遵守 30 天的 Wash Sale 规则（参见 [06_strc_tax_wash_sale.md](06_strc_tax_wash_sale.md)），但 NRA **完全不受此约束**。
- 你可以月月重复操作，不用担心 30 天间隔限制
- 不需要每年 12 月的"31 天空窗期"来斩断洗售链条
- 这大幅简化了执行流程

### 2.4 普通美股股息（非 ROC）参考

如果你持有的其他美股发放的是**普通股息 (Ordinary Dividend)**，则适用中美税收协定：
- 法定预扣税率 = **30%**
- 提交 W-8BEN 后 = **10%**

## 三、中国端的税务义务

中国对税务居民的**全球收入**征税，因此理论上：

| 收入类型 | 中国税率 | 实操情况 |
| -------- | -------- | -------- |
| 股息/利息 | 20% | ROC 本质上不是"股息"，而是"返还本金"，是否征税存在灰色地带 |
| 资本利得（买卖差价） | 20% | 理论上应申报，但目前缺乏对境外券商账户的系统性征管 |
| 外汇出入 | N/A | 换汇额度限制（每人每年 5 万美元），大金额需其他途径 |

> [!IMPORTANT]
> **CRS 信息交换**：IB 会向中国税务机关交换你的账户信息（依据共同申报准则 CRS）。虽然目前中国税务机关对此类信息的实际稽查力度有限，但建议咨询跨境税务专家，确保长期合规。

## 四、IB 端的关键操作清单

1. **确保 W-8BEN 表格有效**
   - 路径：IB Account Management → Tax Forms
   - 申明中国税务居民身份
   - 有效期 3 年，到期需重新提交

2. **每季度检查 Tax Documents**
   - 确认 STRC 分配是否被正确标记为 ROC
   - 若被错误标记为 Ordinary Dividend 并预扣了税款（10%），及时联系 IB 申请退税

3. **保留完整交易记录**
   - 虽然美国端不需要主动报税，但保留完整交易流水
   - 备份年度 Activity Statement，以应对中国端可能的税务审查

## 五、NRA 策略收益总览

| 来源 | 年化收益 | 美国端税负 | 实际落袋率 |
| ---- | -------- | ---------- | ---------- |
| 底仓 ROC 股息 | **11.5%** | 0% | **~100%** |
| Dividend Capture（12次/年） | **~8.5%** | 0% | **~100%** |
| 极端事件加仓（0~2次/年） | **0~10%** | 0% | **~100%** |
| **综合预期** | **~20%+** | **几乎全额免税** | **~100%** |

> [!TIP]
> 对于 NRA，STRC 策略的综合收益几乎可以 100% 落袋。没有 Wash Sale 的执行限制，也没有 cost basis 追踪的复杂度。唯一的变量是中国端未来的税务政策演变。
